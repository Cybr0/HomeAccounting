--------------------------------- Установка // Первичные Настройки ---------------
1) На ПК должна быть установленна Postgresql.
2) Создать пользователя с именем "homeaccountingUser" и паролем "123".
3) Создать базу данных с именем "homeaccountingtest" 
4) Установить бекап из папки backup в эту db.
5) Запустить .exe из папки "HomeAccounting".


----------------------------------- О программе -----------------------------------

Данная программа написана на языке C# c использованием технологии WPF. В проекте используется база данных Postgresql.
